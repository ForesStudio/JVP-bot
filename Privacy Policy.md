# Privacy Policy for Just Voice Bot

## 1. Information We Collect
Just Voice Bot collects the following information:
- **User IDs**: To track individual user statistics
- **Message Counts**: Number of messages sent in text channels
- **Voice Activity**: Time spent in voice channels
- **Server IDs**: To manage server-specific settings

## 2. How We Use Information
The collected information is used for:
- Displaying user statistics (/stats command)
- Generating activity rankings (/top command)
- Server activity monitoring
- Improving bot functionality

## 3. Data Storage
- All data is stored locally in JSON files
- Data is not shared with third parties
- We do not sell or rent any user data
- Data is retained until manually reset by administrators

## 4. Data Control
Server administrators can:
- Reset all statistics using /resetstats command
- Remove the bot to stop data collection

Users can:
- View their collected data using /stats command
- Request data deletion by contacting server administrators

## 5. Data Security
We implement reasonable security measures to protect collected data. However, no method of electronic storage is 100% secure.

## 6. Children's Privacy
The Bot is not intended for users under 13. We do not knowingly collect data from children under 13.

## 7. Changes to Policy
We may update this privacy policy periodically. Continued use of the Bot constitutes acceptance of the updated policy.
